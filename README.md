# Stock Trading Platform

## 📌 Project Description
Stock Trading Platform is a Java console application that simulates a basic stock market where users can buy and sell stocks and monitor their portfolio.

## 🚀 Features
- Display available stocks
- Buy stocks
- Sell stocks
- View portfolio
- Track account balance
- Portfolio value calculation

## 🛠 Technologies Used
- Java
- Object-Oriented Programming (OOP)
- HashMap
- ArrayList

## ▶️ How to Run
1. Open the project in your Java IDE.
2. Compile the program.
3. Run `StockTradingPlatform.java`.

## 📷 Sample Output
Shows market data, buy/sell operations, and portfolio details.
